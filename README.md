# Trove (Proof-of-Concept)

Anonymous, open source, zero-knowledge encrypted cloud storage that provides encrypted vaults based on seed phrases. Simply a proof of concept for now. Used multiple different LLMs to audit the security of the app and to help with problems I got stuck on, such as breaking large files into chunked uploads and upload pause/resume.
